# SPARC Galaxy Rotation Curve Data

## Provenance

This directory contains analysis outputs and rotation curve data from the **SPARC database** (Spitzer Photometry & Accurate Rotation Curves).

**Source:** SPARC Galaxy Rotation Curve Database
**DOI:** 10.5281/zenodo.16284118
**License:** CC-BY-4.0
**Website:** https://astroweb.case.edu/SPARC/

## Required Citation

If you use this data, please cite the SPARC master paper:

**Lelli, F., McGaugh, S. S., & Schombert, J. M. (2016), "SPARC: Mass Models for 175 Disk Galaxies with Spitzer Photometry and Accurate Rotation Curves", The Astronomical Journal, 152, 157**

ADS: https://ui.adsabs.harvard.edu/abs/2016AJ....152..157L/abstract

## Additional Citations

Individual rotation curves in SPARC are compiled from various original sources. If your analysis relies heavily on specific galaxies, please refer to SPARC Table 1 for the original rotation-curve references and consider citing them as well.

## Contents

This directory contains:
- Rotation curve plots for individual SPARC galaxies (PNG format)
- Model fitting results and residual analysis
- Parameter optimization outputs (kappa scans)
- Summary metrics and statistics
